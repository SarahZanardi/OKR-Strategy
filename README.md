# OKR Performance Monitor

Projeto criado a partir do curso **Introducao a Gestao de OKRs para Inovacao**.

## Ideia

O projeto e um monitor interativo para acompanhar objetivos e resultados-chave de um ciclo de inovacao. Ele permite:

- visualizar objetivos de um trimestre;
- acompanhar progresso medio dos resultados-chave;
- filtrar objetivos por status;
- editar textos diretamente na tela;
- adicionar ou remover objetivos e resultados-chave;
- usar um checklist semanal de acompanhamento.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Relacao com o curso

O projeto aplica conceitos centrais de OKR:

- objetivos qualitativos e inspiradores;
- resultados-chave mensuraveis;
- acompanhamento por ciclos;
- check-ins semanais;
- foco em aprendizado, alinhamento e inovacao.
